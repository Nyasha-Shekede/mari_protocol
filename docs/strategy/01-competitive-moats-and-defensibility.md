## Moat #1: Network Effects (The Strongest Moat)

### The Winner-Takes-All Dynamic

**Payment networks have extreme network effects:**
```
1 user = 0 value (can't send to anyone)
10 users = 10 possible connections
100 users = 4,950 possible connections
1,000 users = 499,500 possible connections
1,000,000 users = 499,999,500,000 connections

Value grows exponentially with users
```

**Once Mari has critical mass (1M+ users):**
- New competitor starts with 0 users
- Their network is worthless
- Users won't switch (no one to send money to)
- Competitor dies

**Real-world examples:**
- **WhatsApp**: Killed competitors despite being copied
- **Facebook**: Network effects = unbeatable
- **M-Pesa**: Dominates Kenya despite many copycats
- **UPI**: Dominates India (government-backed network)

**Mari's strategy:**
- Get to 1M users FAST (18-24 months)
- Lock in bank partnerships
- Create switching costs
- Network effects = game over for competitors
